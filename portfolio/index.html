Sai

I build cloud-native data platforms — the kind that move from a sketch on a whiteboard to a CFO checking next quarter's forecast on their phone. Most of that work lives inside Azure: Data Factory for orchestration, Databricks for the lakehouse, Power BI at the edge where decisions get made.

For the last few years I've been doing this at large industrials — Maersk, ADM — where data hygiene is the difference between a clean monthly close and a thousand panicked Slack messages.

Lately the work has shifted. With Databricks' AI/ML stack maturing, the question stopped being "can we get the data to the user?" and became "can we put an agent next to the data?" — that's where I've been investing my time.
