# Hello, I'm Sai.

I'm a Senior Data Engineer and Data & Analytics Technical Lead with nearly a decade of work across cloud-native data platforms.

Most of that work has lived inside the Azure ecosystem — Azure Databricks, Azure Data Factory, Synapse, Power BI — at companies like **ADM** (Finance Record-to-Report), **Maersk** (global logistics analytics), **Ecolab** (Customer Data Management), and **Microsoft** (Worldwide Payment Services).

## What I'm working on

Currently leading the data engineering function for Finance Record-to-Report at ADM, and spending an increasing share of my time on **applied Generative AI** — Databricks Genie Spaces, Vector Search–powered RAG, multi-agent orchestration, and hosting these as Databricks Apps with custom Streamlit UIs.
